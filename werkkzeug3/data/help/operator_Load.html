Load Operator

 

The Load Operator is a leaf. It allows you to link any named object in the document into a tree.

 

Inputs

 

None.

 

Parameters

 

Load

Select the object to link with the Find an op! Dialog ([...] button), choose from the named operators on the current operator page [..] button or simply type a name. The [->] button jumps to the linked operator in the Page Window.

 

Keyboard shortcut

 

l (when adding operators)
