Bricks Operator

 

The Bricks Operator is a leaf. It generates a brickwork-like texture.

 

Inputs

 

None.

 

Parameters

 

Width

Sets the texture width.

 

Height

Sets the texture height.

 

lock

Locks the texture resolution to 100%, overriding the the setting in the Edit menu.

 

Color 0

Defines brick color 0.

 

Color 1

Defines brick color 1.

 

Color Joints

Defines the joints color.

 

Size Joints (vertical/horizontal)

Controls joint width.

 

Count (vertical/horizontal)

Controls the amount of bricks.

 

Seed

A unique identifier for the pseudorandom pattern

 

Single width stones

Controls the maximum share of single width stones

 

Flags

no adjacent sin (on/off): If unchecked, allows adjacent single width stones. If checked, requires a double wide stone between singles.

joints color(on/off): If unchecked, the brick color will bleed into seams. If checked, the seams will be flatly painted in the joints color.

 

Multiply

Controls texture detail level.

 

Side

Controls the vertical row alignment.

 

Color balance

Brick colors are taken from a gradient between Color 0 and Color 1. This parameter affects the gradient curve.
